# Updating things like java ...

## sdkman (sdkman.io)

```bash
# lookup new versions
sdk update

# list them
sdk list java

# install choosen version i.e.:
sdk install java 17.0.8-tem

# remove old version
sdk install java 17.0.7-tem

# same for maven
sdk list maven
sdk install maven 3.9.4
sdk uninstall maven 3.9.3
```
